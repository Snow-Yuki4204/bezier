## A：直接支持我研究假设的

这类文献与“泊车 / 低速车辆路径规划 / A* 与贝塞尔曲线结合优化”最接近，可直接用于支撑“在侧方泊车中引入贝塞尔曲线优化有助于提升路径平滑性、可行性与实时性”这一研究假设。

### 基于搜索与优化算法的自主代客泊车路径规划研究
- 作者：尹禹兮
- 年份：2025
- 研究方法：将自主代客泊车划分为行车和泊车两个阶段，行车阶段采用改进 Hybrid A* 加贝塞尔曲线平滑，泊车阶段采用粒子群优化进行路径规划。
- 核心结论：分层搜索与优化结合的方法能够提升复杂停车场景中的路径规划安全性与计算效率。

### 面向智能停车场的AGV路径规划和路径跟踪
- 作者：吴宇轩
- 年份：2025
- 研究方法：针对停车场 AGV 路径规划，引入转弯惩罚项和 Floyd 思路去冗余节点，并用二阶贝塞尔曲线平滑转弯段，同时设计 MPC 路径跟踪控制器。
- 核心结论：改进 A* 配合贝塞尔曲线平滑可以在保持实时性的同时减少路径长度和转弯频次。

### Automated Parking Planning with Vision-Based BEV Approach
- 作者：Yuxuan Zhao
- 年份：2024
- 研究方法：面向自动泊车任务，提出基于视觉 BEV 局部地图的改进 A* 规划方法，将车辆运动学模型、启发式函数优化、双向搜索、贝塞尔曲线优化和数值优化结合。
- 核心结论：在自动泊车场景中，改进 A* 加贝塞尔曲线优化能够降低计算时间并改善舒适性与实时性表现。

### 基于改进A*算法和三阶贝塞尔曲线融合的路径规划分析
- 作者：朱文亮，王志鹏
- 年份：2025
- 研究方法：通过设计新的启发式函数和权重系数改进 A* 搜索，并结合三阶贝塞尔曲线进行路径平滑。
- 核心结论：A* 与贝塞尔曲线融合能够同时减少搜索节点并改善路径质量。

### A Cost-Effective Approach to Smooth A* Path Planning for Autonomous Vehicles
- 作者：Lukas Schichler，Karin Festl，Selim Solmaz，Daniel Watzenig
- 年份：2024
- 研究方法：在 A* 中引入曲率约束与平滑性代价函数，并利用预计算运动原语与两阶段平滑规划生成可驾驶路径。
- 核心结论：经过平滑和曲率约束增强的 A* 能生成更适合类汽车平台执行的低曲率路径。

## B：提供方法论依据的

这类文献不一定直接针对侧方泊车，但能为你的论文提供“贝塞尔曲线如何建模、如何约束、如何与搜索或优化耦合”的方法论依据。

### 五阶贝塞尔曲线的机器人路径平滑算法设计
- 作者：周洋，靳兴来，黎鑫，胡敏
- 年份：2025
- 研究方法：基于分段五阶贝塞尔曲线进行折线路径平滑，建立带线性约束的凸优化模型，并以迭代二次规划求解。
- 核心结论：分段高阶贝塞尔曲线可以在满足安全约束的同时生成曲率连续的平滑路径。

### Spatio-temporal Motion Planning for Autonomous Vehicles with Trapezoidal Prism Corridors and Bézier Curves
- 作者：Srujan Deolasee，Qin Lin，Jialun Li，John M. Dolan
- 年份：2022
- 研究方法：在 `S-L-T` 三维时空空间中构建梯形棱柱时空走廊，并在其中进行分段贝塞尔曲线优化。
- 核心结论：比立方体走廊更宽松的时空走廊能扩大贝塞尔优化解空间并保证连续时间安全性。

### A Dynamic Motion Planning Framework for Autonomous Driving in Urban Environments
- 作者：Yuncheng Jiang，Xiaofeng Jin，Yanfei Xiong，Zhaoyong Liu
- 年份：2019
- 研究方法：用三次样条平滑道路中心线，再用五阶贝塞尔曲线生成曲率连续轨迹，并将动态规划解耦为横向空间规划与纵向速度规划。
- 核心结论：五阶贝塞尔曲线适合用于生成更符合人类驾驶风格且曲率连续的车辆轨迹。

### 融合A*算法与贝塞尔曲线的井下车辆全局路径规划方法
- 作者：付杨，张雾琳，白龙
- 年份：2025
- 研究方法：提出 A* 搜索与贝塞尔曲线平滑相结合的全局路径规划方法，通过算法协同兼顾路径搜索适应性和轨迹平滑性。
- 核心结论：A* 与贝塞尔曲线的协同设计可以同时兼顾路径搜索效率与轨迹平滑性。

### 面向城市场景的智能车辆轨迹规划算法研究
- 作者：陈俊毅
- 年份：2025
- 研究方法：构建时空联合轨迹规划框架，使用五阶贝塞尔曲线生成参考线，并引入时空混合 A* / ARA* 与后端优化。
- 核心结论：贝塞尔曲线适合作为复杂车辆轨迹规划中的参考线与平滑表示工具。

### 基于改进混合A*的联合轨迹规划方法
- 作者：李妍婷，崔俊杰，王天惠，史国强
- 年份：2025
- 研究方法：先用改进混合 A* 在三维时空空间中生成初始轨迹，再构建时空走廊并用分段贝塞尔曲线进行后处理优化。
- 核心结论：搜索生成粗轨迹、贝塞尔曲线完成后端平滑优化的联合框架具有较好的灵活性与安全性。

## C：与我研究对象相关但方法不同的

这类文献与“车辆路径规划 / 泊车或低速机动 / 特定交通场景轨迹规划”相关，但核心方法路线与“侧方泊车中的贝塞尔曲线优化”不同，适合放在相关工作中作为对照背景。

### 改进双向人工势场法的自动驾驶路径轨迹规划
- 作者：李强，丁伟奇，梁洪波，张新闻
- 年份：2025
- 研究方法：提出双向人工势场法与二次 A* 相结合的路径规划方法，并用贝塞尔曲线平滑最终路径。
- 核心结论：双向人工势场法能够缓解局部极小值与目标不可达问题，并缩短复杂环境中的路径长度。

### 基于贝塞尔曲线的进藏大型货车避障轨迹多目标规划方法
- 作者：叱干都，赵阳，王建军，孙战丽
- 年份：2025
- 研究方法：提出三维两阶段四次贝塞尔曲线轨迹规划方法，将避障和回正分阶段处理，并与车速规划耦合成多目标动态优化问题。
- 核心结论：在特殊道路条件下，多阶段贝塞尔轨迹规划可以兼顾大型货车避障的安全性、稳定性和实时性。

### Path-Tracking Hybrid A* and Hierarchical MPC Framework for Autonomous Agricultural Vehicles
- 作者：Mingke Lu，Han Gao，Haijie Dai，Qianli Lei，Chang Liu
- 年份：2024
- 研究方法：提出 Path-Tracking Hybrid A* 与分层 MPC 相结合的框架，通过新的代价函数和启发式函数减小对参考路径的偏差。
- 核心结论：面向农业车辆的 Path-Tracking Hybrid A* 与分层 MPC 能显著提升路径贴合度和实时避障能力。
