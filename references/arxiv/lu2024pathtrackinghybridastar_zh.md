# 面向农业车辆的路径跟踪式 Hybrid A* 与分层 MPC 框架

原文：`lu2024pathtrackinghybridastar.pdf`  
作者：Mingke Lu，Han Gao，Haijie Dai，Qianli Lei，Chang Liu  
年份：2024

## 摘要译文

本文提出一种路径跟踪式 Hybrid A* 规划器，并将其与分层模型预测控制框架结合，用于农业车辆路径平滑。目标是在跨垄作业时尽量减小相对于参考路径的偏差，从而提升作业效率、减少作物和土壤损伤，同时满足曲率约束和整车碰撞避免要求。论文的主要贡献有三点：第一，提出 Path-Tracking Hybrid A* 算法，通过设计新的代价函数和启发式函数，在非完整约束下同时实现参考路径贴合、曲率受限和整车避碰；第二，提出一种在线重规划扩展，使系统在遇到未知障碍物时能够实时规避，并利用剪枝技术提高计算效率；第三，设计分层 MPC 框架，利用线性化 MPC 为非线性求解器提供热启动，在实时满足车辆约束的同时提高求解收敛性。基于真实农田数据的仿真表明，该方法在安全性、路径贴合度、计算速度和实时避障方面都优于基线方法。

## 1. 引言

随着机器人技术发展，农业车辆在监测、喷洒、收获和除草等任务中展现出巨大潜力。路径规划因此成为农业机器人中的基础能力之一。已有研究大量关注地形起伏、覆盖规划、多任务执行和能量效率等问题。

然而，跨垄作业这一任务仍然研究不足。在这类场景中，车辆需要穿越垄沟，同时紧密跟踪预先给定的参考轨迹。哪怕很小的横向偏差，也可能造成作物破坏、土壤压实或作业效率下降。因此，这类任务的目标不只是“找到一条可行路径”，而是“找到一条尽可能贴近参考路径、同时满足整车避碰和曲率约束的平滑轨迹”。

作者指出，传统平滑方法虽然在通用机器人中已有应用，但通常更强调几何平滑或曲率连续性，而不是最小化对参考轨迹的偏离；此外，很多方法使用点质量模型，忽略整车尺寸，这在狭窄农田场景中是不够的。

## 2. 相关工作

论文首先回顾了 Hybrid A* 在路径规划中的作用。Hybrid A* 兼顾了网格搜索与连续空间可行性，近年来已与可视图、安全走廊、时间弹性带、聚类和动态优化等技术结合，主要被用作前端搜索器。但作者认为，它在“路径平滑”这一角色上仍有进一步潜力。

其次，作者回顾了路径平滑方法的三类典型思路：

- 基于插值的方法，如贝塞尔曲线和 B 样条；
- 基于特殊几何曲线的方法，如 Clothoid、Dubins、Reeds-Shepp；
- 基于优化的方法，如 PSO、ACO 等。

这些方法虽然能生成平滑路径，但在农田跨垄场景中，要么不强调对参考路径的严格贴合，要么无法处理整车碰撞约束。

## 3. 系统建模

论文建立了农田环境模型和农业车辆模型。环境中存在垄沟、障碍物以及需要严格遵守的作业轨迹，因此车辆不能仅以点模型规划，而必须把整车轮廓纳入碰撞检测。车辆运动满足非完整约束，因此轨迹生成与跟踪均需考虑转向和曲率限制。

## 4. 方法

### 4.1 Path-Tracking Hybrid A*

作者提出的 Path-Tracking Hybrid A* 不仅要求搜索出一条从起点到终点的可行轨迹，还要求轨迹尽可能贴近参考路径。为此，论文重新设计了代价函数和启发式函数，使搜索时显式惩罚相对于参考轨迹的偏差，同时考虑曲率约束和整车避碰。

与普通 Hybrid A* 相比，这种做法会让规划结果不再只是“可达”，而是“沿着参考线路以最小偏差到达”，更符合农业作业需要。

### 4.2 在线重规划与剪枝

为了应对突发障碍物，作者在该规划器基础上扩展出在线重规划机制。当检测到新障碍物时，算法能够迅速在局部重新搜索一条满足约束的替代路径。为了控制实时开销，论文使用剪枝技术减少无效分支，从而提高求解效率。

### 4.3 分层 MPC

在轨迹跟踪阶段，作者设计了分层 MPC 框架。上层更关注参考轨迹跟踪，下层则负责满足车辆动力学与整车碰撞约束。为了提高非线性优化问题的求解效率，作者先用线性化 MPC 生成一个热启动解，再交给非线性求解器继续细化。这一策略在保持精度的同时改善了收敛速度。

## 5. 仿真结果

作者在真实农田数据上进行了对比实验，结果显示：

- 该方法生成的轨迹更贴近参考路径；
- 曲率约束能够被严格满足；
- 整车碰撞避免效果优于只考虑点质量模型的方法；
- 在线重规划能应对突发障碍物；
- 分层 MPC 在实时性和求解稳定性方面优于直接求解复杂非线性问题的基线方案。

## 6. 结论

本文提出的核心思想，是把“参考路径跟踪”直接融入 Hybrid A* 的搜索过程，再通过分层 MPC 完成后续平滑与控制。论文表明，这一框架特别适用于农业车辆跨垄作业，因为它不仅能避障和满足曲率约束，还能尽量减小对作物和土壤有害的横向偏差。
